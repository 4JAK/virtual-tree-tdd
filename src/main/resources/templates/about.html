About the project

Utilizing a Java based Spring framework planT'DD allows users to learn the priciples and technics of test driven development. Several styles of engaging interactive questions and visual examples allow users a simple and fun way to build their knowledge. The program was built using the knowledge attained by the team during thier 14 week full time bootcamp. Including a Java written back-end it also uses Java-Script, and Ajax to populate pages using Thymeleaf, Css, and HTML to create a smooth cohesive feel.
