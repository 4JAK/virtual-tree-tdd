About the project

Utilizing a Java based Spring framework, planT'DD allows users to learn the priciples and techniques of test driven development.

Several styles of engaging interactive questions and visual examples give users a simple and fun way to build their knowledge.

The program was built using the knowledge attained by the team during their 14 week full-time bootcamp.

Including a Java back-end, it also uses JavaScript and Ajax to populate pages, as well as Thymeleaf, CSS, and HTML to create a smooth cohesive feel.
